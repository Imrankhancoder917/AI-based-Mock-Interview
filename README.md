## 🚀 Overview

InterviewForge is an AI-powered mock interview platform that simulates real-world technical and behavioral interviews using resume-aware and job-description-aware questioning.

The platform analyzes a candidate's resume and target job description, generates adaptive interview questions, evaluates responses using AI, supports voice-based interactions, and produces detailed performance reports with actionable feedback.

### 🎯 Key Highlights

- Resume & Job Description Analysis
- AI-Generated Interview Questions
- Adaptive Follow-Up Questioning
- Voice-Based Interview Support
- AI Answer Evaluation & Scoring
- Performance Analytics & Reports
- PDF Report Export
- Secure Authentication System
- Full-Stack Flask Architecture

## 💡 Why This Project?

Most interview preparation platforms ask generic questions.

InterviewForge personalizes the entire interview process by understanding:

- Candidate background
- Skills
- Experience
- Target role requirements

This creates a realistic interview experience similar to actual recruiter and technical interviews.
